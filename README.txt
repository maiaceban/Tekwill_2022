# Tekwill-2022 Java Fundamentals
![cover-site](https://user-images.githubusercontent.com/98480742/175118815-fea274d2-2fcf-41ec-b7f1-43f0a2bd1dca.jpg)

Conform datelor Oracle, Java este unul dintre cele mai utilizate limbaje de programare la nivel global și rulează pe 3 miliarde de telefoane mobile, 
1 miliard de calculatoare și 45 de miliarde de mașini virtuale. Aceste cifre sunt rezultatul a doar 8 milioane de dezvoltatori Java la nivel global 
care inovează zi de zi lumea înconjurătoare. În Republica Moldova, 25% din joburile deschise pentru angajarea dezvoltatorilor software necesită cunoștințe de Java.

Module:
I. Java Overview;
II. Java OOP. Classes, Objects, Data and Methods;
III. Java API;
IV. Inheritance. Handle Exceptions;
V. Software Testing. RESTful Applications;
VI. Introduction to Databases. JDBC. JPA.
